# 02 模块 / 组件 / 功能完整索引

> 全量映射：模块 → 包 → 组件（类）→ 功能 → 源文件。用于快速定位。

## 1. pimono-demo

| 模块 | 组件 | 主要功能 | 源文件 |
|---|---|---|---|
| 入口 | `PiMonoDemoApplication` | env 探测、REPL 命令（exit/history/status/clear/tools/help）、装配 | `pimono/PiMonoDemoApplication.java` |

| 配置 | `MCPConfig` | 声明式 stdio/http 配置 | `pimono/mcp/MCPConfig.java` |
| UI | `ConsoleUI` | JLine3 ANSI REPL、命令解析 | `pimono/ui/ConsoleUI.java` |

## 4. 关键配置参数

| 参数 | 默认值 | 来源 |
|---|---|---|
| `MAX_TOOL_ROUNDS` | 50 | pimono `AgentCore` |
| `maxIterations` | 50 | agentscope `Agent` |
| `MAX_CONTEXT_MESSAGES` | 50 | pimono `ContextManager` |
| 上下文压缩阈值 | 40 条 | agentscope `ContextCompressionMiddleware` |
| 上下文压缩保留 | 10 条 | agentscope `ContextCompressionMiddleware` |
| `DEFAULT_BUDGET` | 500_000 tokens | agentscope `ReplyBudgetControlMiddleware` |
| 执行超时 | 30 s | agentscope `CodeExecutionManager` |
| 文件大小上限 | 10 MB | agentscope `FilePermissionConfig` |
| LLM `max_tokens` | 4096 | agentscope `ChatModel` |
| LLM `temperature` | 0.7 | agentscope `ChatModel` |
| OkHttp connect | 30 s | 两个模块 |
| OkHttp read | 120 s | 两个模块 |

## 5. 环境变量

| 变量 | 用途 | 适用 |
|---|---|---|
| `OPENAI_API_KEY` / `DASHSCOPE_API_KEY` / `ANTHROPIC_API_KEY` / `DEEPSEEK_API_KEY` | 至少一个 | 两模块 |
| `MODEL_PROVIDER` | 留空自动探测 | 两模块 |
| `MODEL_NAME` | 例 `gpt-4o-mini` / `glm-5.1` | 两模块 |

| `MCP_SERVERS` | 格式 `command:arg1,arg2` | 两模块 |
| `WORKSPACE_DIR` | 默认 `workspace/` | agentscope |
| `REPLY_BUDGET` | token 预算 | agentscope |
